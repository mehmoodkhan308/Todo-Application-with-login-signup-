# ✅ Todo Application with Login & Signup

Welcome to my **Todo Application with Authentication System** 🚀  
This project is built using **HTML, CSS, and JavaScript** and includes a complete login/signup system with a dashboard-based todo app.

---

## 📌 Live Demo

🌐 https://todo-application-with-login-sinup.vercel.app

---

## 💻 Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- LocalStorage (for data handling)

---

## 📁 Project Features

### 🔐 Authentication System
- User Signup page
- User Login page
- Basic form validation
- Data stored using LocalStorage

### 📝 Todo Application
- Add new tasks
- Delete tasks
- Mark tasks as complete
- Simple dashboard UI

### 📊 Dashboard
- User-friendly interface
- Task management system
- Clean UI layout

---

## 📂 File Structure

- `index.html` → Login Page  
- `sign.html` → Signup Page  
- `dashboard.html` → Todo Dashboard  
- `app.js` → Login logic  
- `sign.js` → Signup logic  
- `dashboard.js` → Todo functionality  
- `style.css` → Login page styling  
- `signupstyle.css` → Signup styling  
- `dashboard.css` → Dashboard styling  

---

## 🎯 Features

- Login & Signup system (frontend-based)  
- Todo CRUD operations  
- LocalStorage-based data saving  
- Multi-page application  
- Clean and simple UI  
- Beginner-friendly project  

---

## 🚀 How to Run

1. Clone the repository  
2. Open `index.html` in browser  
3. Signup a new user  
4. Login and access dashboard  
5. Start managing tasks 🎉  

---

## ⚠️ Note

This project uses **LocalStorage only**, so data is saved in browser (not server/database).
